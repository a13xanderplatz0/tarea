# Sistema de Gestión de Recursos Educativos

Este proyecto implementa una base de datos para gestionar recursos educativos, incluyendo:
- Gestión de usuarios
- Recursos educativos
- Simulacros de exámenes
- Sistema de ranking

## Estructura de la Base de Datos

- **Usuario**: Almacena información de usuarios
- **Recursos**: Gestiona materiales educativos
- **Simulacros_Examenes**: Control de exámenes
- **Ranking**: Sistema de clasificación

## Tecnologías
- Oracle Database
- PL/SQL

## Instalación

1. Ejecutar script de creación de tablas
2. Ejecutar inserciones de datos de prueba
3. Verificar con consultas de ejemplo
